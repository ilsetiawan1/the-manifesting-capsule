# 🔒 The Manifesting Capsule

A frictionless, anonymous, and time-locked web application designed for the modern generation to secure their text manifestations, goals, and reflections into the future. 

### 🚀 Core Pillars
* **Pure Text Sanctuary:** Focused solely on the power of words and personal declarations.
* **Frictionless Engagement:** Fully anonymous architecture with no registration required, utilizing secure HttpOnly Cookie-based Access Keys.
* **Absolute Privacy:** Time-locked messages are structurally protected and selectively queried directly from the database level to prevent client-side inspection leaks.

### 🛠️ Tech Stack
* **Frontend:** Next.js 15+ (App Router), Tailwind CSS, Shadcn UI, Framer Motion.
* **Backend & Database:** Next.js Server Actions, Prisma ORM, Zod, PostgreSQL, NanoID.
